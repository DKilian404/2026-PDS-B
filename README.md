# Projects in Data Science (2026)
### Preliminary Assignment - GROUP B
  In this repository, you will find a PDF file with a summary of the analysis performed on the images, masks, and data frames of skin lesions. 
  In addition, the ‘results’ folder contains the Jupyter Notebook file with the images generated to support the information mentioned in the PDF.
  Finally, in the annotators folder, there is a CSV file with the 117 annotations made by each member of the group.

### Instructions
  In order to correctly execute the jupyter notebook file, you must clone the entire repository and execute the file normally.
  If you only want to download the jupyter file, you must also download the Merged.csv file located in the ‘annotators’ folder and then indicate the path of that file in the      first cell of the jupyter file, more precisely within the ‘annotations’ variable.

  Once you run the first cell, the programme will ask you for the path to the image folder, then the path to the mask folder, and finally the path to the metada.csv file.         Please note that we are group B, so these files must be filtered beforehand.
